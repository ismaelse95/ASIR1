# Servidor DHCP

Aquí podemos ver el proceso en el que una máquina pide una ip a un servidor DHCP.

DCHP DISCOVERY
	- Difusion enlace
	- Difusion IP
	- Puerto 67/udp

DCHP OFFER
	- MAC Cliente
	- Difusión IP
	- Puerto orig: 67, Puerto dest: 68

DHCP REQUEST
	- MAC servidor
	- IP servidor
	- Puerto 67/udp
